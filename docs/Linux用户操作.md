title: Linux 用户操作

## 基本操作

### 添加用户

```bash
useradd -m 用户名
```

### 设置密码  

```bash
passwd 用户名
```

### 删除用户

```bash
userdel  -r  用户名
```

### 删除用户目录

```bash
rm -rf 用户名
```

### 切换当前用户

```bash
su 用户名
```
